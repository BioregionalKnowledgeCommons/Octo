# Protocol Library

This directory is the local staging area for shared pattern and protocol artifacts derived from reviewed practices.

The source-of-truth entities still live in the KOI graph and vault. This folder is for workflow support, publication manifests, and future local library materialization.

Shared protocols here should be treated as coordination patterns, not universal recipes. A protocol should stay linked to the practices, patterns, and case studies it came from, along with fit conditions, rights/consent constraints, and contraindications.

Before a protocol is used in another place, it should normally be re-instantiated as a local playbook with adaptation notes for that bioregion rather than copied verbatim.
